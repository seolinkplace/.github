## SEOLinkPlace

Open source link placement marketplace — self-hosted, modular, production-ready.

Built with **Laravel 13**, **Filament 5**, and a modular monolith architecture.

- [seolinkplace.com](https://seolinkplace.com) — live platform
- [community](https://github.com/seolinkplace/community) — open source core (MIT)

### Stack
`Laravel 13` · `PHP 8.3` · `Filament 5` · `MariaDB` · `Redis` · `Go`

### Community Edition includes
- Multi-role auth (clients, webmasters, performers)
- Site registry with domain verification
- Admin panel (Filament 5)
- Support ticket system
- Public blog
- 11 languages

**Enterprise Edition** with campaigns, billing, payments and more available separately.
